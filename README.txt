Written in python 3.8.6

HOW TO USE:

1. use pip install -r dependencies.txt to install dependencies.

2. create a .env file and insert the following informations:

DISCORDBOT_TOKEN = Your_Discordbot_token
AWS_ACCESS_KEY_ID = your_aws_access_key_id
AWS_SECRET_ACCESS_KEY = your_aws_secret_access_key
REGION_NAME = your_aws_region
SERVER_INSTANCE_ID = your_server_instance_id

3. Finally, run bot.py
